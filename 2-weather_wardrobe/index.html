Computer Science Education Week

Weather Wardrobe

One of the fun things about technology is being able piece together multiple components to create something new. In this activity we'll create a program texts you in the morning to tell you how to dress for the weather that day.

Note: This activity is for Apple machines.

Implementing on a PC requires additional development that will not be covered here.

Agenda

  1. Create a burner Google account
  2. Write and test script
  3. Send a message to our phone
  4. Set up crontab on computer

1. Create "burner" Google account

You definitely don't want to use your official Google account for this project, so let's do this:

  1. Go to google.com and sign up for a new account so that we can use Gmail. The username and password don't matter, but it should be a good password. Write it down somewhere.
  2. Enable that account to use less secure apps. You can do that by going to your settings and choosing: My Account > Sign-in & security > Allow less secure apps: ON

Confirm that you can log into the account, then log out. You'll be using this account to send emails/texts to yourself.

2. Write and test script

  1. You're going to have a program that runs on your computer, and we need a place to put it. The Desktop works fine, or your Documents folder.
  2. Launch the TextEdit app on your Apple laptop, open up Preferences and change the default format to "Plain text." We're going to be editing the program that will make this application work.
  3. Right-click or control-click to download this file and move it onto your Desktop.
  4. With TextEdit, open up the file you downloaded. Then edit the code to make the following changes:
    1. put your burner email address in the appropriate space
    2. put your password in the appropriate location
    3. pout your custom weather messages in there
    4. Put your Poly email address in the appropriate location. (We'll change this to your cellphone number after testing.)
    Then save the file.
  5. Run the script
    1. Launch a Terminal app on your phone
    2. Type which python in the terminal and copy the message that gets displayed. Then use that information to run the program as shown:
      % which python /opt/anaconda3/bin/python % /opt/anaconda3/bin/python /Users/$USER/Desktop/weather_wardrobe.py
      If the program works correctly, you should see a message displayed in your terminal window. You should also receive an email at your Poly email.

3. Sending email to your phone

  1. Cellphone carries usually allow emails to be sent to your phone using a specially-formatted email address.
  2. Start with your cellphone number. I'll use 626-845-1235 and take out the dashes: 6268451235.
  3. Add the appropriate domain name as specified by your cellphone carrier:
    • Verizon: 6268451235@vtext.com
    • T-Mobile: 6268451235@tmomail.net
    • AT&T: 6268451235@txt.att.net
  4. Using TextEdit, change the email in the program from your Poly email to your cellphone number as described above. Save this new version of the program.
  5. Using the Terminal, try running the program again:
    % /opt/anaconda3/bin/python /Users/$USER/Desktop/weather_wardrobe.py
    6. You should get text message on your phone with the message from the program!

4. Set up crontab

Here's the tricky part: getting your script to run automatically every morning!

  1. Back in the Terminal again, copy the command that you used to run the program. Then...
  2. Open up the crontab program that we can use to run programs at any specified day and time.
    % export EDITOR=nano % crontab -e
  3. Enter the following line into the crontab:
    00 12 * * 1-5 /opt/anaconda3/bin/python /Users/$USER/Desktop/weather_wardrobe.py
    This will try to automatically run the program at 12:00. After entering the line, hit ctrl-O to save the file, Y to accept the filename, then ctrl-X to exit the text editor.
  4. Wait a moment and see if it works!
  5. To run the program in the morning, change the "00 12" to "00 7" for 7am, and you'll get your text message with the weather every weekeday morning!
